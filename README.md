# TCP-Client-Server
用c++编写的TCP连接 

# 使用步骤

1.编译

~~~ basH
cd TCP-server-client-c++
mkdir build
cd build
cmake ..
make
~~~
2.运行

~~~bash
cd bin
./TCP_server
#在另外开一个终端执行：

./TCP_client
~~~
